# Inventory Management System (SQL Project)

## Description
This project demonstrates a relational database design using MySQL.

## Features
- Supplier → Product → Inventory relationships
- Normalized schema
- Views and Stored Procedures
- Ready for Power BI integration

## Tools Used
- MySQL (XAMPP)
- SQL
- Draw.io (ER Diagram)

## Database Objects
- Tables: customers, orders, order_items, supplier, product, inventory
- View: inventory_summary
- Procedure: GetExpensiveProducts
             GetLowStock
